## Project 1: Lab 1 – Confusion Matrix & Regression Metrics
### Description
Binary classification and regression metrics implemented in Python as calculator-style exercises.
### Dataset
Synthetic values provided for TP, FP, TN, FN, SSR, SSE, etc.
### Steps Performed
1. Calculated Accuracy, Precision, Recall, F1 Score.
2. Visualized metrics with bar chart.
3. Computed R², MSE, Cook’s Distance.
4. Applied simple linear regression prediction.
### Results
- Accuracy: 0.82  
- Precision: 0.86  
- Recall: 0.83  
- F1 Score: 0.84  
- R² = 0.70 (model explains 70% variance).  
### Tools Used
Python, Matplotlib, NumPy
### Conclusion
Introduced core evaluation metrics for classification and regression models.
